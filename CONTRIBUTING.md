# Contributing to CBRN-Sentinel

Thank you for your interest in contributing to CBRN-Sentinel!

## How to Contribute

1. **Found a bug or have a suggestion?** Please open an issue to discuss it before submitting a PR.
2. **Submitting Code:**
   - Fork the repository.
   - Create a feature branch (`git checkout -b feature/your-feature`).
   - Ensure you format your code properly and run existing tests.
   - Create a Pull Request with a clear description of your changes.

## Development Setup

1. Ensure you have Python 3.10+ installed.
2. Create a virtual environment: `python3 -m venv .venv && source .venv/bin/activate` (or `.\.venv\Scripts\activate` on Windows).
3. Install dependencies: `pip install -r requirements.txt`.
4. Run tests: `pytest` (if applicable) to ensure everything works locally.

We value contributions that enhance AI safety and biosecurity!
